# EZ AD — Easy Active Directory

A modern web-based management hub for Active Directory. Replaces ADUC and GPMC with a clean dark UI accessible from any browser on your network.

**Features**
- OU tree with users, computers, and groups
- Create and edit AD users and groups
- GPO list with searchable settings (parsed from `Get-GPOReport`)
- RSoP — see which GPOs apply to any user
- JWT-based authentication
- Runs as a Windows Scheduled Task (SYSTEM account)

---

## Requirements

- Windows Server 2016+ (or Windows 10/11)
- [Node.js 18+](https://nodejs.org/en/download)
- [Git](https://git-scm.com/download/win)
- RSAT: Active Directory + Group Policy PowerShell modules

---

## Install

Open an **elevated PowerShell prompt** and run:

```powershell
irm https://raw.githubusercontent.com/dalt0n0/EZAD/main/install.ps1 | iex
```

The script will prompt for a web UI password, then clone the repo, install dependencies, build, register a startup task, and open the firewall.

**With options:**
```powershell
& ([scriptblock]::Create((irm https://raw.githubusercontent.com/dalt0n0/EZAD/main/install.ps1))) -Password "P@ss!" -Port 8080 -Username sysadmin
```

Once installed, browse to **http://\<server\>:3000**

---

## Update

Pull the latest code and rebuild without touching credentials:

```powershell
C:\EzAD\install.ps1 -Update
```

Or directly from GitHub:

```powershell
& ([scriptblock]::Create((irm https://raw.githubusercontent.com/dalt0n0/EZAD/main/install.ps1))) -Update
```

---

## Manage the service

```powershell
Stop-ScheduledTask  -TaskName EzAD   # stop
Start-ScheduledTask -TaskName EzAD   # start
```

---

## Uninstall

```powershell
C:\EzAD\install.ps1 -Password x -Uninstall
```

Then optionally remove files:

```powershell
Remove-Item C:\EzAD -Recurse -Force
```

---

## Configuration

Credentials and port are stored in `C:\EzAD\.env.local`:

```
EZAD_USERNAME=admin
EZAD_PASSWORD=YourPassword
EZAD_SECRET=<auto-generated>
PORT=3000
```

Edit this file and restart the task to apply changes.

---

## Install options

| Parameter | Default | Description |
|---|---|---|
| `-Password` | *(prompted)* | Web UI password |
| `-Username` | `admin` | Web UI username |
| `-Port` | `3000` | Port to serve on |
| `-InstallPath` | `C:\EzAD` | Where to install |
| `-SkipFirewall` | — | Skip firewall rule creation |
| `-Update` | — | Pull latest and rebuild |
| `-Uninstall` | — | Remove task and firewall rule |
